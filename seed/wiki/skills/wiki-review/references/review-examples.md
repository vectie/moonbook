# Review Examples

## Good approval note

`Approved: the run added a maintained source page, refreshed the related entity page, and the wording stays within the evidence recorded in the source packet.`

## Good revision note

`Revise: the evidence is useful, but the concept belongs under an existing page and should not create a new ontology fragment.`

## Good rejection note

`Rejected: the result only touched administrative pages and does not justify durable promotion.`

## Bad review note

`Looks fine to me.`
