# Query Note Examples

## Good saved-query note

```
# Why did the research run stop at administration?

## Question
Why did the last cross-project research run fail to produce durable knowledge pages?

## Answer
The control plane and book bootstrap succeeded, but the ingest provider returned generic completion output without substantive source, entity, concept, or query artifacts.

## Cited pages
- wiki/sources/moonclaw-source.md
- wiki/synthesis/overview.md
- wiki/history/debug-journal.md

## Uncertainty
The wiki still lacks complete runtime coverage for the provider-task boundary.

## Follow-up
- inspect raw bootstrap packets
- inspect provider result normalization
```

## Weak note to avoid

```
# Research status

It seems blocked.
```
